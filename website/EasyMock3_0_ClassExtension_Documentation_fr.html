EasyMock Class Extension permet de générer des Mock Objects à partir de classes.

EasyMock Class Extension est à partir de maintenant considérée comme dépréciée.

Les mocks de classes peuvent et doivent être fait directement à l'aide d'EasyMock classique. La documentation concernant les mocks de classes a été intégrée à celle d'EasyMock.

Pré-requis

EasyMock Class Extension nécessite Java 5 ou supérieur, EasyMock 3.0,

Utilisation

Au passage vers EasyMock et EasyMock CE 3.0, vos projets devraient fonctionner comme avant (à l'exception d'avertissement de dépréciation). Toutefois, vous devrez les recompiler.

Migration

Migrer d'EasyMock CE vers EasyMock 3.0 est simple. Vous devez retirer la dépendance à EasyMock CE et remplacer tous les imports vers 

org.easymock.classextension.*

par des imports vers 

org.easymock.*

ce qui veut concrètement dire d'enlever .classextension de votre code.
